# Install for Antigravity

Use the generic `AGENTS.md` style adapter unless your Antigravity setup has a different project-instruction filename.

```bash
cp adapters/antigravity/AGENTS.md /path/to/project/AGENTS.md
cp PROJECT_MEMORY_WORKFLOW.md /path/to/project/PROJECT_MEMORY_WORKFLOW.md
cp -R templates /path/to/project/.project-memory-templates
```

If `AGENTS.md` already exists, merge the adapter section instead of overwriting it.
