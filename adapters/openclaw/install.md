# Install for OpenClaw

If OpenClaw supports project instruction files, place `OPENCLAW.md` or merge its contents into the project instruction file.

Suggested per-project setup:

```bash
cp adapters/openclaw/OPENCLAW.md /path/to/project/OPENCLAW.md
cp PROJECT_MEMORY_WORKFLOW.md /path/to/project/PROJECT_MEMORY_WORKFLOW.md
cp -R templates /path/to/project/.project-memory-templates
```

If OpenClaw uses a different instruction filename in your environment, copy the adapter content into that file.
